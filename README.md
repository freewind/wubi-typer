Wubi Typer
==========

用于练习五笔字根。

打开后，将会显示500个五笔字根，在键盘上输入相应的键即可。结束后，将会显示速度与正确率。

注：
- 打开页面后就开始计时
- 每个字根只有一次输入机会，不能回退或重输
- 把鼠标放在相应的字根上，会显示对应的键位
- 需要自行翻页
- 时间、速度与正确率等，在最下方显示

Run:

```
npm install
npm start
```

Then visit <http://localhost:8080>.

For dev:

```
npm install
npm install -g http-server

webpack -w
http-server .
```
